# Premier League Club Owners Report

AI-generated ownership appraisal of all 20 Premier League clubs, researched 1 July 2026.

## Origin

Prompted by a thread on [Forest Forum](https://forestforum.co.uk/index.php?threads/financial-fair-play-ffp.45929/page-553) where fans were posting ownership lists, others calling them useless, and someone mentioning AI — so I decided to just make an AI report.

## What it is

A single self-contained HTML file (`premier-league-ownership-appraisal-all-20-clubs-with-radars.html`) scoring all 20 clubs across **8 dimensions** on a 0–100 scale.
