# Premier League Club Owners Report

AI-generated ownership appraisal of all 20 Premier League clubs, researched 1 July 2026.

## Origin

Prompted by a thread on [Forest Forum](https://forestforum.co.uk/index.php?threads/financial-fair-play-ffp.45929/page-553) where fans were posting ownership lists, others calling them useless, and someone mentioning AI — so I decided to just make a proper AI report.

## What it is

A single self-contained HTML file (`premier-league-ownership-appraisal-all-20-clubs-with-radars.html`) scoring all 20 clubs across **8 dimensions** on a 0–100 scale:

- Sporting value added
- Governance & leadership
- Financial stewardship
- Capital commitment
- Football operations
- Infrastructure & revenue development
- Stability & adaptability
- Fan & community relations

Each club gets a weighted **Ownership Benefit Score**, a radar chart, and a sourced written appraisal.

## Methodology

Evidence window: last 5 completed seasons. Sources prioritised by: company filings → official statements → high-quality journalism. Scores are evidence-weighted judgements, not exact measurements.
